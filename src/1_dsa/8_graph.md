# Graph
- data structure used to represent relationships between objects
- It consists of:
    - Vertices (or Nodes) → Represent entities
    - Edges → Represent connections between entities
- The edges can be weighed or not
- e.g.
    - not weighed
    ```
    A — B
    |   |
    C — D
    ```
    - weighed
    ```
    A —0.5— B
    |       |
    1       2
    |       |
    C —0.3— D
    ```
